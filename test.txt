git clone  - скачивание репозитория на пк (нужна ссылка репозитория)
git status - статус
cd - переход из папки в папку
git add . - Добавить всё
git add "имя" - Добавить какой-то файл
git commit -m "комментарий" - Добавить коммит
git push - Загрузить файл на сервер

git diff "имя файла" - Показывает изменения в файле (строки кода)
